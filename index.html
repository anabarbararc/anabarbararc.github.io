Sentiment Analysis

Skills: Web Scrapping, ETL, Pandas, NLP, Matplotlib

The main idea is to analyse the sentiment (positive or negative) of sentences using the polarity feature. The data was acumulated by web scrapping 1000 twitts containing the terms 'covid', 'climate change' and 'cripto'. Then it was transformed into a Pandas dataframe. The polarity for each tweet was analysed and plotted in a histogram.
